Touchless Hand Sanitizer Dispense
Description:
The Touchless Hand Sanitizer Dispenser with Infrared Sensor is a project aimed at providing a hygienic and convenient solution for hand sanitization in various environments. This project incorporates an infrared sensor-based system that detects hand movements and dispenses hand sanitizer without the need for physical contact, minimizing the risk of spreading germs and viruses.

Technologies used:
Infrared Sensor, suction motorpump, battery , small pipe, container

Multi-Axis Servo Control using Joystick
Description:
The Multi-Axis Servo Control using Joystick project is a versatile system that allows precise and intuitive control of multiple servo motors using a joystick input. This project provides a user-friendly interface and enables smooth and simultaneous control of servo motors along multiple axes, making it suitable for applications such as robotics, automation, camera control, and remote-controlled systems.

Technologies used:
2- Servo motor, joystick, Arduino, jumper wires, breadboard/PCB

Obstacle Avoiding Robot
Description:
The Obstacle Avoiding Robot is an intelligent and autonomous robotic system designed to navigate through its environment while avoiding obstacles in its path. This project incorporates sensors and control mechanisms to enable the robot to detect obstacles, make real-time decisions, and maneuver around them to ensure safe and efficient traversal.

Technologies used:
Servo motor, ultrasoonic sensor, Arduino, jumper wires, breadboard/PCB, two motors with wheel , motor shield
